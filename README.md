# TGProject_carbonads
[![building](https://img.shields.io/travis/rust-lang/rust.svg?style=flat-square)](building)

## 功能
提供Web学习网站

## 启动
1. 下载所需依赖  
```bash
npm install
```
2. 启动服务器
```bash
npm run serve
```

## 命令
- 启动开发模式，同时会启动下面两个命令
```bash
npm run dev
```
- 启动对Sass进行监视，如果sass文件有改动，自动编译
```bash
npm run watch-sass
```
- 运行node服务器
```bash
npm run serve
```
